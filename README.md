The goal of this project is to procedurally generate a configurable landmass. A terrain heightmap, plus a water layer, plus a number of other possible ideas:
- biomes
- civilizations
  - settlement placement
  - roads/trade routes

In theory once we have these static elements we can then layer in a time tick element and simulate events like:
- population
- economy
- environment / seasons
- war
- 
...but these elements might be just a *little bit* complicated, so we'll see what ends up happening.
